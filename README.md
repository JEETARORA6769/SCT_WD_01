# Interactive Navigation Menu

A responsive and interactive navigation menu built using **HTML**, **CSS**, and **JavaScript** as part of my Web Development Internship.

## 🚀 Features

- 📌 Fixed navigation bar
- 🎨 Navigation bar changes color while scrolling
- 🖱️ Hover effects on navigation links
- 📱 Fully responsive design for mobile devices
- ☰ Hamburger menu for smaller screens
- ❌ Auto-close mobile menu after selecting a navigation link
- 🔍 Active navigation link highlighting based on the current section
- ⚡ Smooth scrolling between sections

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📸 Preview

The website contains five sections:

- Home
- About
- Services
- Projects
- Contact

Each section can be accessed through the fixed navigation bar, which updates dynamically as the user scrolls through the page.

Create an interactive navigation menu that:

- Changes style when the page is scrolled.
