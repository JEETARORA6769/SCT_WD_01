# 🌐 Interactive Navigation Menu

A fully responsive and interactive navigation menu built using **HTML, CSS, and JavaScript** as part of my **Web Development Internship at SkillCraft Technology**.

This project demonstrates the fundamentals of responsive web design, DOM manipulation, event handling, and interactive user interfaces. The navigation bar remains fixed while scrolling, changes appearance based on the scroll position, highlights the active section, and adapts seamlessly to different screen sizes using a responsive hamburger menu.

## 💡 Functionality

### Fixed Navigation

The navigation bar remains fixed at the top of the page, ensuring easy access while scrolling through different sections.

### Scroll Effect

When the user scrolls down the page, JavaScript dynamically adds a CSS class that changes the navbar's background color and size.

### Active Navigation Highlight

The current section is detected using `offsetTop`, `offsetHeight`, and `window.scrollY`, allowing the corresponding navigation link to be highlighted automatically.

### Responsive Design

The website adapts to different screen sizes using CSS Media Queries. On smaller screens, the navigation transforms into a hamburger menu.

### Mobile Navigation

The hamburger icon toggles the navigation menu using JavaScript. After selecting a menu item, the menu automatically closes for a better user experience.
